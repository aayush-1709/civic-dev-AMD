# CivicDev – Citizen Issue Reporting Platform

## 📌 Overview
CivicDev is a web-based platform where citizens can report civic issues like potholes, garbage collection problems, water leakage, or streetlight failures.  
The system helps authorities track and resolve these issues efficiently.

---

## 🚀 Features

### User
- Register and Login
- Report civic issues
- View reported issues
- Track issue status

### Admin
- View all complaints
- Update issue status
- Manage reports
